# Convertidor a PDF 
Script creado en Python para convertir archivos por lotes a PDF. 

Disponibles para:
- Microsoft Office Visio
- Microsoft Office Powerpoint

**Nota**: Debe ser ejecutado dentro de la misma carpeta con los archivos a convertir.
